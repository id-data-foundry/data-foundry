Publishing information

Zenodo is a free repository to share scientific data. Built and developed by researchers, to ensure that everyone can join in Open Science. It allows researchers to upload and share research datasets.

When you publish your project you create a "deposit" to upload data to Zenodo. The deposit is the publication itself. It contains title, date, authors, DOI, description, and much other optional metadata. Once created, we can upload files to the deposit.

Data Foundry will use the Zenodo API, which is available at https://zenodo.org/api/ and requires authentication through an access token. The access token is generated in Zenodo Settings > Applications > Personal access tokens. Access tokens are generated with permissions called "scopes", and we need the "deposit:write" scope to create deposits and upload files. After defining a name for the access token and selecting the scope, the site will generate a key, i.e., a random string that we will use to authenticate with the API. Please treat the key as a secret because anyone with the key can upload files on your behalf.
