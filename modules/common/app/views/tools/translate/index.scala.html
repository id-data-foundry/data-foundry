Translate

This tool provides AI translations with adjustable styles. Simply select your desired source and target languages and a style, and start typing in the left pane. The text is automatically sent to our locally hosted Large Language Model (LLM) which processes the translation and sends the result back.

Translated with Apertus-8b: We use the Apertus-8b model for translations. This is a highly capable 8-billion parameter open-weights model fine-tuned for multilingual tasks and nuanced language understanding. Because it runs locally on Data Foundry servers, your data remains completely private and secure.
