Disclaimer

The media-to-text processing is performed using machine learning models and without correctness guarantees. Please check the output carefully and do consider giving us feedback about the transcription quality.

Image, video and audio files that are uploaded to this service will only be processed on this server (in the Netherlands), and no data will be transmitted outside of the EU (this is important for GDPR compliance). The files will be deleted immediately after processing in the following steps for audio/video: (1) audio stream extraction from uploaded file and transcoding to temporary file, (2) speech-to-text processing, (3) output transcribed text and deletion of all files.

We are using open-source technology to provide this service. For speech-to-text functionality, we use the Parakeet v3 model. For the text extraction from images, we use the open-source Tesseract OCR component with Python bindings, and Python image processing libraries. For PDF processing, we use OpenDataLoader.
