Activity Logger

This page allows you to configure an activity logger with activities specific to a dataset in one of your projects. By categorizing activities into "Direct Event" and "State" types, you can tailor how data is logged and visualized for your research or project.

To begin, please select a project and then choose the specific dataset you wish to configure. Your selections will load existing activities and enable the configuration options below.

1. Select project and dataset

The activity logger will attempt to log into a dataset in this project. To keep things organized, select a participant that you want log activities for. If you don't want that, make sure the dataset is switched to "open participation".

2. Define direct event activities

These are activities that occur at a specific point in time, without a duration. Examples include discrete actions like "Drinking Water," "Button Press," or "Feedback Submitted." Define names for these single, instantaneous events that you wish to log for the selected dataset.
