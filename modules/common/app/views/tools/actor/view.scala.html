Script examples

This is a simple example that prints two lines to the output: the first includes two different data items from the OOCSI message, the second just prints the entire message object, which is great for understanding which data items are included. This can be installed on any channel. More information in the scripts documentation. 

          
// DF.print can print strings with data items
DF.print("Color data (" + data.color + "), position data (" + data.position + ")");

// DF.print can also print the entire OOCSI message
DF.print(data);

This example shows how to selectively log new data into an IoT dataset based on a condition (data.temperature > 34) and for a specific device. We also supply an activity or event marker "temperature_alert" which can be used for filtering. This can be installed on any channel. More information in the scripts documentation. 

          
// assuming that the message contains some data with property temperature and a device id
if(data.temperature > 34) {
  // log an event for the given device id with 
  // the activity "temperature_alert" and data containing the high temperature
  DF.eventData.log(data.device_id, 'temperature_alert', {temperature: data.temperature});
}

This example shows how to update a user profile (an entity dataset) with a message value (in case the message matches a condition). This can be installed on any channel. More information in the scripts documentation. 

          
// assuming that the message contains some data with property temperature and a user_id
if(data.temperature > 34) {
  // first retrieve the profile values
  var profile = DF.entity.get(data.user_id);
  // set the new value
  profile.temperature = data.temperature
  // then update the profile
  DF.entity.update(data.user_id, profile);
}

This example shows how to receive a message and inform all researchers in the project about it (in case the message matches a condition). This can be installed on any channel. More information in the scripts documentation. 

          
// assuming that the message contains some data with property temperature
if(data.temperature > 34) {
  // in case of a temp value greater 34 send a Telegram message to all researchers in the project
  DF.telegramResearchers('Hey, temperature is quite high today (' + data.temperature + '), perhaps check on participant 1?');
}

This is a very simple chatbot example that show how to respond to an incoming Telegram message. You need to use "telegram" for the channel when you install this. More information in the scripts documentation. 

          
// use this to log the entire message
// DF.print("message: " + data.message)

// respond to message with a particular word or expression
if(data.message.contains("happy")) {
  DF.telegramReply("Great, tell me more about why you are happy.")
} 
// respond to a message with an arith. expression
else if(data.message.match("[0-9+*/-]+")) {
  DF.telegramReply(eval(data.message))
} 
// catch all response
else {
  DF.telegramReply("No idea, continue.")
}
