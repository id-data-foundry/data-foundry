Data Logger

A data logger is an easy way to log on-the-go data into a dataset of your choice. To create a data logger you need to select (1) a project, then (2) a dataset and then (3) a device ID to log with. Finally this page generates a unique link and QR code for you to start logging. Alternatively, you can log with Sensor Logger, for which we create a configuration below as well.

1. Select a project and dataset

The data logger will attempt to log into a dataset in this project. If the dataset is switched to "open participation", you can freely enter a device name. If not, you need to select from existing devices below.
