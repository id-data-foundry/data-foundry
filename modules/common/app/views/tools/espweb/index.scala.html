ESP Web Tools

1. MicroPython Web Installer

This page allows you quickly and easily install MicroPython on ESP32 boards directly from the browser. Simply connect the board to an USB port, click on the button "Connect", select the device and have MicroPython running on your board! *If you are using the Making Sense of Sensors board, hold the boot button while inserting your usb cable*

This functionality is not available because your browser does not support Web Serial. Open this page in Google Chrome or Microsoft Edge instead (but not on your iOS device).

If you don't see your ESP device in the list of devices to choose, you might need to install the drivers. Also, you might need to hold the boot button while inserting your usb cable.

Troubleshooting
  • __ Check that your USB cable supports data and not just charging.
  • __ Check that you have selected the right USB serial device (and not your mouse or keyboard).
  • __ Check that you have a working ESP32 (test with another computer?).
