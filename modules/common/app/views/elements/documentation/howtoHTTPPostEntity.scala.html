  • Java

    We have a Java library that simplifies accessing the Entity dataset. Download the precompiled binary (jar file) or check out the source bundle.

    Depending on your Java programming environment, for instance, Processing or an IDE like Eclipse or IntelliJ, you need to import the library first before you can use it. In Processing, you need to download the jar file linked above and then put it into a "code" folder next to the Processing pde file that you want to use the library with. After that restart Processing (for safe measure) and use the library in your code. 

               
// import the library
import nl.tue.id.datafoundry.*;
import java.util.Map;
// fill in the Data Foundry server URL, just the domain (add port if needed)
DataFoundry df = new DataFoundry("@routes.HomeController.index().absoluteURL(request)");
// create dataset access
DFDataset entity = df.dataset(@dataset.getId(), "@dataset.configuration(Dataset.API_TOKEN, "TOKEN")");
// or: create access directly for a specific item in the dataset
DFDataset item = df.dataset(@dataset.getId(), "@dataset.configuration(Dataset.API_TOKEN, "TOKEN")").id("userX").token("nosecrets");
// add an item
item.data("temperature", 34).data("door", "open").add();
// get an item
Map<String, Object> itemData = item.get();
String doorState = (String) itemData.get("door");
// doorState = "open"
// update an item
item.data("temperature", 32).update();
// delete an item
item.delete();
