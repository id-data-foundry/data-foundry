Combining datasets Combining information from various data sources is a common and useful operation. ### Joining along rows You may want to gather similarly structured data in one large dataset, for instance when your app exports one file per day, with exactly the same data but (probably!) different values. An easy way to join such files into one large one, is by using the pandas function **concat()**. See the simple example below. # %% [python] # heartrate on Monday df_monday = pd.DataFrame([['8:00:00', 60],['13:00:00',90],['20:00:00',80]], columns = ['time', 'heartrate']) df_monday.head() # %% [python] # heartrate on Tuesday df_tuesday = pd.DataFrame([['8:00:00', 50],['13:00:00',90],['20:00:00',70]], columns = ['time', 'heartrate']) df_tuesday.head() # %% [python] # heartrate on Monday and Tuesday df_mo_tue = pd.concat([df_monday, df_tuesday]).reset_index() df_mo_tue.head() # %% [markdown] ### Joining along columns Alternatively, you might consider joining different types of information into one richer dataset. For example, by combining heartrate data with location data recorded by your phone, you might find answers to questions like "Am I more stressed at school than at home?". For this combination to be meaningful, a **key** is needed, that is one or more features that are found in both original datasets. Location, time, person ids are examples of features that usually work as keys. The recommended way to do perform this join is by using the pandas function **merge()**, which provides many options and is well documented. The following code will combine the stepcount and phoneusage data into one new dataset, which could be used to study the link between someone's physical activity and phone behavior, or just to keep all relevant data in one place. # %% [python] df_all = df_steps_agg.copy() df_all = pd.merge(df_all, df_phoneusage_agg, left_on=['Participant','Date','Hour'], right_on = ['Participant','Date','Hour'], how='inner') df_all.head() # %% [markdown] This is a rather compact dataset, still it is not straightforward to find any correlations just by looking at the numbers.
